IMAP Generator Page: Create User Flow

Base direction is now the calmer module page. Sync conflict stays hidden until a manual sync finds differences, and previous IMAP passwords open from a modal instead of living on the page.

A

Inline Create User Panel

If fuzzy search finds no good match, a create-user form expands inside the page below the resolver. It keeps the current IMAP inputs visible while creating the user.

B

Create User Modal

If no match is found, open a focused modal to create the user, then return to the page and continue saving the generated IMAP credential.
